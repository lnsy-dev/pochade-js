# Pochade-JS
## For Writing JS with passion


There are some frameworks for writing careful enterprise software. This is not one of them. 

Pochade-js is for writing quick, vanilla modules and static websites. It includes context-engineering helpers. 

It compiles CSS and Web Workers for quick, easy deployments. 

It has an assets folder that is served at / and then copied over to dist at build time for deployment.

It uses standard ES6 modules. It ships with the dataroom-js library for ease creating custom HTML Elements. 


## Using Pochade-js

Create a new project with Pochade-js with the command: 

```sh
npx pochade-js name-of-new-project
```
